   Funktionsbeschreibung:
   Dieses Modul Übergibt seine Parameter an den Prozess ELECTRONICAL_LOAD.
   Im Prozess ELECTRONICAL_LOAD werden dann die Ansteuerung durchgeführt.
   Das Messergebnis wird per Returnparameter aus diesem Modul weiter 
   an das Modul M_AUSWERTEN gegeben.
