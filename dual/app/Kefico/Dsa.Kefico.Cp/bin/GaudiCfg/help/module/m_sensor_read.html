   Funktionsbeschreibung:
   Dieses Modul uebergibt den die Sensornummern und den Sendestring
   and den Sensorserver der diese an die Interfacerechner versendet. 
   Die von dem Interfacerechner enthaltenen Werte werden in dem
   Array zurueckgegeben.
   
   Beschreibung der Modulübergabeparameter:
1 PRUEFSCHRITT
   GAUDI Prüfschritt
   ------------------------------
2 POSITION
   PAV- Position
   ------------------------------
3 LAUFENDE
NUMMER
   Es wird die 
   ------------------------------
4 ART
DER ADRESSEN
   SIB oder IR
   IR  - Interfacrechner Adresse
   SIB - SensorInterfaceBaugruppe
   ------------------------------
5 ADRESSEN
   Arrayvariable in der fuer jede Adresse
   den Wert 1 (senden) oder 0 (nicht senden)
   gespeichert ist.
   ------------------------------
6 ERGEBNISMODE
   WERTELISTE: die Ergebnisse werden in ein double  Array gespeichert
   OKLISTE:    die Ergebnisse werden in ein integer Array gespeichert
   ZUENDLISTE: die Ergebnisse werden in ein integer Array gespeichert
   OERFOLG:    das Ergebnis wird in eine einfache Variable gespeichert
   ------------------------------
7 ERGEBNIS
   Rueckgabe der angeforderten Ergebnisse.
   -------------------------------
8 FEHLERFLAG
   0 = ok
   1 = nicht Ok (illegale Parameter oder Fehler
                 beim Empfangen)
