CCS Modul   P_DHP
 		 
   ---------------------------------------------------------------------
   Funktionsbeschreibung:
   Das Modul dient der Steuerung von Messgeräten für Dichtheits- und
   Durchflussprüfungen. Die Funktionsweise ist aktionsorientiert.
   Im Modul wird eine auszuführende Aktion, die u.U. Parameter
   haben kann, angegeben. Der Prozess hinter P_DHP entscheidet dann
   je Aktion, ob und wenn ja, wie mit dem Messgerät kommuniziert wird.
   
   DHP_TESTER
   Symbolischer Name des Messgerätes. Er muss in der DHP_CNF konfiguriert.
   
   DHP_AKTION
   Schlüsselwort für eine auszuführende Aktion.
   
   DHP_TIMEOUT
   Maximale Zeit in ms, die die Aktion dauern darf.
   
   PARA1 - PARA4
   Parameter für eine Aktion. Hängt von der angegebenen Aktion ab.
   
   RETURN1 - RETURN3
   Rückgabewerte für eine Aktion. Hängt von der angegebenen Aktion ab.
   
   ERRORFLAG
   Fehlerflag, muss eine GAUDI-Variable sein
   0		Kein Fehler
   1		Fehler
