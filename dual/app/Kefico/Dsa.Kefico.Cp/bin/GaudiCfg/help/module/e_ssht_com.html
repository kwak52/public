   Aufrufparameter laut Modglobal.
   
  < 1> PRUEFSCHRITT
  < 2> POSITION
  < 3> SSHT_NR          SSHT-Nr z.B. SSHT_1
  < 4> FU_SSHT_COM      MOVIE, STOPMOVIE, GRAB, ADDIMAGES,...             
  < 5> SSHT_ADR         Adresse Register, EEPROM, Destination Bildeinzug
  < 6> SSHT_VALUE       z.B. Wert/Werte die zu programmieren sind
  ( 7) SSHT_LENTH       Laenge
  ( 8) SSHT_MODUS       L/S/INIT Lesefunktion
  ( 9) RETURN_WERT
  (10) MG_ERROR

   Frame Grabber Karte durchzufuehren
