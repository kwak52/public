   Funktionsbeschreibung:
   uk_write:
      Ausgabe eines beliebig langen Strings an ein Geraet. Der String wird 
      an den Prozess UK ueber die Messagequeue geschickt, der diesen dann
      an das Geraet ausgibt.
   uk_writeReadFile:
      Sendet oder empfaengt eine Datei ueber ftp.
