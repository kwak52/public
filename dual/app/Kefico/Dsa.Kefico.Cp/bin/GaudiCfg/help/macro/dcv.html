DCV [V] (Us_low, Us_high)

MaxSignalAmplitude [V]

IntegrationPeriod [s]

Description of Function

Integrating DC voltage measurement with AUTO triggering.

Testing Points

Test point + Reference point

Parameters

MaxSignalAmplitude

Largest amplitude occurring on the signal. Can be used to establish the measurement range or to select a suppressor circuit.

IntegrationPeriod

Period over which the signal is integrated. In the event of superimposed interference signals, it is advisable to select a multiple of the interference signal period as the integration period.

Standard value: period duration of supply voltage.

Example
