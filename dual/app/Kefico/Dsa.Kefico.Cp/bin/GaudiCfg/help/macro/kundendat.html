CCS Standardfunction   KundenDat [Str]
 

KundenDat [Str]

Data_Type [Str]

Data_Format [INT]

Description of Function

This function is used to request general information on the testing station such as date of manufacture. The result returned is always a string.

Testing Points

None

Parameters

Data_Type

Specifies the type of information required. For permissible values refer to table below.

Data_Format

There are various possible formats for the representation of some types of information. This parameter is used to specify the desired format. The format is always specified by an integer. For permissible values refer to table below.
