RDCVTrg_2 [V]

Description of Function

Test read function for internally or externally triggered integrating DC voltage measurement.
The function RDCVTrg_2 may only be used in conjunction with the function DCVTrg_2 IDH_DCVTrg_2 (Mode s).

Testing Points

None

None
