RDCVTrg [V] (RUab, RUhigh, RUlow)

Description of Function

Test read function for internally or externally triggered integrating DC voltage measurement.
The function RDCVTrg may only be used in conjunction with the function DCVTrg IDH_DCVTrg (Mode s).

Testing Points

None

Parameters

None
