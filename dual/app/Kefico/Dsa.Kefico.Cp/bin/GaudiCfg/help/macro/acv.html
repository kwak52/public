ACV [V]

MaxSignalAmplitude [V]

IntegrationPeriod [s]

Description of Function

Measurement of effective level with automatic triggering.

The execution time for this testing function is relatively long since, due to the AC coupling of the testing instrument it is essential to wait for the output stage to complete its settling time before recording a measurement. In order to avoid this problem, wherever possible a combination of ACVInit IDH_ACVInit and RACV IDH_RACV should be used.

Testing Points

Test point + Reference point

Parameters

MaxSignalAmplitude

Largest amplitude occurring on the signal (relates only to the alternating component). Can be used to establish the measurement range or to select a suppressor circuit.

IntegrationPeriod

Period over which the signal is integrated.
In the event of superimposed interference signals, it is advisable to select a multiple of the interference signal period as the integration period.

Standard value: period duration of supply voltage.
