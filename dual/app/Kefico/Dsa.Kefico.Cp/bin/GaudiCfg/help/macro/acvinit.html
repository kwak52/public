ACVInit []

MaxSignalAmplitude [V]

IntegrationPeriod [s]

Description of Function

Sets up an ACV measurement. Due to the AC coupling of the testing instrument, the output stage has to be allowed to complete its settling time before a measurement can be recorded. The ACVInit function can be invoked as soon as all parameters required for the voltage measurement have been applied. The voltage to be measured is then recorded using the function RACVIDH_RACV. IDH_RACV Other testing stages can be carried out between the two function calls (provided they do not affect the voltage to be measured) so that the settling time can be utilized. This function does not return a value.

Testing Points

Test point + Reference point

Parameters

MaxSignalAmplitude

Largest amplitude occurring on the signal (relates only to the alternating component). Can be used to establish the measurement range or to select a suppressor circuit.

IntegrationPeriod

Period over which the signal is integrated.
In the event of superimposed interference signals, it is advisable to select a multiple of the interference signal period as the integration period.

Standard value: period duration of supply voltage.
