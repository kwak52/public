DCVTrg_2 [V]

For definition and description of function: see DCVTrg (there are no aliases however)

Notes

The definition of a second testing function makes it possible to distinguish between two voltmeters in the PAV. The use of two voltmeters can help to shorten the testing time if there are frequent voltage measurements with differing measurement ranges. One instrument can then be configured for the high measurement range and the other for the low measurement range. In that way, the time required to change the measurement range on each occasion can, in many cases, be saved.

On testing stations which only have one voltmeter, this function is identical with DCVTrg. In those cases, DCVTrg_2 is simply an alias.

The functions DCVTrg and DCVTrg_ 2 should be used sensibly in the PAV, even if the testing system on which the control unit is to be tested only has one voltmeter, as this simplifies subsequent conversion. DCVTrg should be the preferred option for low voltages and DCVTrg_ 2 for high voltages (especially shut-off peaks).

Test stages with a DCVTrg measurement must not be nested in test stages which contain a DCVTrg_2 measurement (or vice versa) as this would require two voltmeters and a standard testing station only has one voltmeter. Nesting is permissible only in justifiable exceptional cases if agreed with the unit carrying out the test.
