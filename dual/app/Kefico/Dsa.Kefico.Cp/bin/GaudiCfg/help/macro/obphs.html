OBPHS []

Edge [p/n]

Timeout [s]

Description of Function

This function is used to perform the OBP handshake between the testing station and the control unit. It can not be executed until the handshake pin has been initialized by means of the function OBPInit1 IDH_OBPInit1 . If a malfunction occurs during the OBP handshake, the test is normally aborted.

This first step sets the level of the ECU_Continue signal according to the specified signal edge. The testing system then waits for the appropriate edge of the TS_Continue signal.

The basis for the function OBPHS is the procedure defined by the OBP Standard Working Party.

This function does not return a value.

Testing Points

None

Parameters

Edge

Specifies the signal edge polarity for the handshake with the control unit. This relates to the TS_Continue signal (p = positive edge, n = negative edge of the TS_Continue signal). In order to force the desired edge at the ECU_Continue pin, the first time the function OBPHS is invoked, the voltage level at the ECU_Continue input on the control unit must be suitably initialized beforehand.

Timeout

If the handshake does not take place within the period specified for the timeout, the complete test is aborted.

Standard value: 1s

Example

OBPHS

n,1
