Legende

Legend_Number [INT]

Other Parameters (any number of other parameters separated by commas)

Description of Function

This function acts as a placeholder for new functions that have not yet been incorporated into the Standard. It must be described in the legend together with all its testing parameters.

Note: this function may only be used in justified special cases as it may complicate or even prevent automated conversion.

Testing Points

Dependent on function.

Parameters

Legend_Number

Reference to a section of the Legend in which the function is explained in more detail. Different legend functions can be distinguished within a PAV by different legend references.

Other Parameters

List of user-defined parameters which - as with the other testing functions - are separated by commas. The function and its parameters must be described in the legend.

Example:

Legende

12,PAR1,PAR2
