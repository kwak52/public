ACI [A]

MaxSignalAmplitude [V]

Test_Duration [s]

Test_Method [Str]

Description of Function

Measurement of current for periodically variable signals (sine-wave, square-wave, triangular, trapezoidal)

Testing Points

The control unit pin at which the current is to be measured is specified as the test point and the reference point.

Parameters

MaxSignalAmplitude

Largest amplitude occurring on the signal. Can be used to establish the measurement range or to select a suppressor circuit.

Test_Duration

Duration of the test necessary so that sufficient periods can be analyzed.

Test_Method

î = Peak value

I< = Arithmetical mean

Ieff = Effective value

Example

ACI

1, 0.1, Ieff
